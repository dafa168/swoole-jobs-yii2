## swoole-jobs-yii2

* 基于swoole-jobs的yii2 demo项目
* 教大家如何整合yii2和swoole-jobs

## 文档

* git clone https://github.com/kcloze/swoole-jobs-yii2.git
* composer install -vvv --profile
* 修改config/swoole-jobs.php配置

## 管理服务

* php swoole-jobs start|stop|exit

## 增加测试任务到队列

* php yii jobs/add
